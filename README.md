# 알라딘 mcp 서버 빌드
`cd aladin-mcpserver && npm run build`

# 웹서버 start
`cd server && npm run dev`<br><br>
.env 파일에서 Anthropic Key 추가해줘야함 (ANTHROPIC_API_KEY={api 키값})<br>
.env 파일에서 서버 포트 추가해줘야함 (SERVER_PORT=3000)

# 페이지 띄우기
`npm run dev`
