# 알라딘 mcp 서버 빌드
`cd aladin-mcpserver && npm run build`

# 웹서버 start
`cd server && npm run dev`
.env 파일에서 Anthropic Key 추가해줘야함

# 페이지 띄우기
`npm run dev`
